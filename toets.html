Het Mobiele Landschap

Wat is het verschil tussen native apps en cross platform apps?

  1. Cross platform app werkt met een webview, native apps niet.

  2. Cross platform app heeft doorgaans betere performance

  3. Cross platform app en native apps zijn één en hetzelfde.

  4. Cross platform app ondersteunt meerdere platform, native is single platform.

Wat is het verschil tussen een native app en een hybrid app?

  1. Native app ziet er op elk OS hetzelfde uit, hybrid app niet.

  2. Een native app performed meestal beter dan een hybrid app.

  3. Native app gebruikt retina display, hybrid app relatieve pixels.

  4. Een native app schrijf je in binary, een hybrid app heeft tekst format.

Cordova/Ionic Native

Wat is Cordova?

  1. Een Cloud platform met native controls voor je Hybrid app.

  2. Een cross platform manier om HTML voor een specifiek OS te genereren.

  3. Een open source framework met JavaScript wrappers van native sensors.

  4. Een manier om met Ionic een cross platform app te maken.

Welk van de volgende sensors kun je benaderen via Ionic Native?

  1. Bluetooth

  2. de Camera

  3. GPS

  4. Al bovenstaande.

  3. Cross platform app ondersteunt meerdere platform, native is single platform.

Wat is het verschil tussen een cross platform app en een hybrid app?

  2. Cross platform app compileert naar native controls, hybrid app runt een JavaScript parser en HTML en CSS renderers.

  3. Cross platform app kan alle native sensors benaderen, hybrid app alleen sensors die JavaScript begrijpen.

  4. Cross platform app vereist dure commerciele IDE’s, hybrid is open source en gratis.

Wat is het verschil tussen een native app en een hybrid app?

  2. Native app gebruikt retina display, hybrid app relatieve pixels.
